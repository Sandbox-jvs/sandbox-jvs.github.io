UDP

User Datagram Protocol This is a lightweight transport protocol that prioritizes speed over reliability as data packets can be lost, duplicated, or arrive out of order during transmission. UDP does not retransmit lost packets or guarantee packet delivery. This protocol is commonly used for streaming and online gaming purposes. When we say that this protocol is connectionless, we mean that no handshake between the devices occur, as further discussed below.

TCP

Transmission Control Protocol TCP is best used when accurate and dependable transmission of data is required. Unlike UDP, TCP is a connection-oriented protocol in that it does go through the handshake process between communicating devices and hosts. However, it also requires more overhead, meaning that it requires more work and data to make communication happen. Although TCP is slower than UDP due to its overhead, it ensures data delivery through acknowledgments and retransmissions.

A handshake is the process by which two devices establish a connection and agree to communicate before data transmission can begin. It is a three-step process in whcih the source device (your computer, for example) makes a request for a connection, the destination device acknowledges the request, and finally, the source device confirms the acknowledgment. Acknowledgements are messages sent by the receiving device responding to confirm that data or connection requests were successfully received. For instance, suppose a user wants to enter an online chat room. Once the chat application asks TCP to establish a connection:

  1. TCP sends a connection request to the destination host, typically a server
  2. The server sends an acknowledement that the request has been received
  3. The source device confirms the the acknowledement, completing the connection so that data transmission can begin

A server is just a host -- a computer or device connected to the network -- that runs software designed to provide services to other devices (in this case, a computer that has the chat server software)
