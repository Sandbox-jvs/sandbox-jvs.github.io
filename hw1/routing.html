Routing

  • Determines the path data is moved between networks
  • Uses routing algorithms to determine the best path
  • Occurs in routers throughout the network
  • Is a Control Plane function
  • Occurs in the Network Layer
  • Routes statically by using manually configured routes
  • Routes dynamically using protocols - RIP, OSPF, BGP
  • Determines the entire end-to-end path before forwarding can occur
  • Anaglogy: Maps out the highways needed to reach a destination (Home -> Universal Studios)

Forwarding

  • The actual moving of the data packets from an input port to an output port
  • Is a local, per-router function
  • Is a Data Plane function
  • Occurs in the Network Layer
  • Uses forwarding tables mapped by routing
  • Uses longest prefix matching
  • Uses packet header information to determine the output port
  • Analogy: Making the actual turns following the highway mapped by the GPS (router)
